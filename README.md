# Astra File Monitor (AFM)

Централизованный полнотекстовый поиск по пользовательским файлам и автоматическое выявление конфиденциальных данных.

## Быстрый старт

```bash
git clone https://github.com/Fedoroff00/afm-server.git
cd afm-server
sudo bash deploy.sh
После установки откройте http://<ваш-IP> и войдите под учётной записью, созданной при установке.

Установка агентов
В веб-интерфейсе перейдите в «Машины» → «Добавить агента», скопируйте команду и выполните на целевой машине.

Управление триггерами
В разделе «Настройки» добавляйте/удаляйте ключевые слова. При их обнаружении в файлах создаются инциденты.

REST API агента (порт 9090)
GET /metrics – метрики

GET /status – состояние агента (JSON)

POST /scan – запустить сканирование

GET /file?path=<путь> – скачать файл

Обновление
bash
cd afm-server
git pull
sudo bash deploy.sh
